# Mixed Pattern Recognition Drill

Use this when you want interview-style practice without knowing the topic in advance.

For each problem:

1. Read the prompt only.
2. Write the likely pattern before coding.
3. Write the expected time and space complexity.
4. Implement.
5. Compare your pattern guess with `PATTERNS.md`.
